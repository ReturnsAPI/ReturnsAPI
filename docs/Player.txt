## Static Methods

<a name="get_local"></a>
```lua
Player.get_local() -> Player or Instance
```

Returns the Player instance of this game client,  
or an invalid Instance if they do not exist.  

**Parameters**  
None

<br><br>

---

## Instance Methods

<a name="equipment_get"></a>
```lua
player:equipment_get() -> Equipment or nil
```

Returns the player's current equipment.  
Always `nil` for non-player actors.  

**Parameters**  
None

<br><br>

<a name="equipment_set"></a>
```lua
player:equipment_set(equip) -> nil
```

Sets the player's equipment.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`equip` | Equipment | The equipment to set. <br>If `-1`, removes equipment.

<br><br>
