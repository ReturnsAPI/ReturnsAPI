## Static Methods  

<a name="bind"></a>
```lua
Script.bind(func) -> Script  
```

Binds a Lua function to a GML script function and returns it.

**Parameters**  
Parameter | Type | Description
| - | - | -
`func` | function | The function to bind.

<br><br>

<a name="wrap"></a>
```lua
Script.wrap(script) -> Script  
```

Returns a Script wrapper containing the provided script.

**Parameters**  
Parameter | Type | Description
| - | - | -
`script` | RValue or Script wrapper | The script to wrap.
