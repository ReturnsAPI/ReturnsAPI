## Static Methods  

<a name="add"></a>
```lua
RecalculateStats.add(fn) -> nil  
```

Registers a function for stat recalculation.

**Parameters**  
Parameter | Type | Description
| - | - | -
`fn` | function | The function to register. <br>The parameters for it are `actor, api`.

<br><br>

<a name="remove_all"></a>
```lua
RecalculateStats.remove_all() -> nil  
```

Removes all registered stat recalculation functions from your namespace.

Automatically called when you hotload your mod.
