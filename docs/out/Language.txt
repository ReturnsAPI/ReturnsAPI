## Static Methods  

<a name="translate_token"></a>
```lua
Language.translate_token(token) -> string  
```

Returns the actual text of the localization token in the current language,
or `token` if none is found.

**Parameters**  
Parameter | Type | Description
| - | - | -
`token` | string | The localization token (e.g., `item.barbedWire.name`)

<br><br>

<a name="register_autoload"></a>
```lua
Language.register_autoload(env) -> string  
```

Registers a mod to autoload language files in a `language` folder.

Automatically called on `.auto()` import.

**Parameters**  
Parameter | Type | Description
| - | - | -
`env` | table | The environment table of the mod to register.
