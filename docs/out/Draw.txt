## Static Methods  

<a name="rectangle"></a>
```lua
Draw.rectangle(x1, y1, x2, y2, [outline], [color]) -> nil  
```

Draws a rectangle.

**Parameters**  
Parameter | Type | Description
| - | - | -
`x1` | number | The left coordinate.
`y1` | number | The top coordinate.
`x2` | number | The right coordinate.
`y2` | number | The bottom coordinate.
`[outline]` | bool | If `true`, the shape is drawn as an outline. <br>`false` by default.
`[color]` | color | The color of the shape. <br>Uses the current draw color by default.

<br><br>
