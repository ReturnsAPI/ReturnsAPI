## Static Methods  

<a name="sprite"></a>
```lua
EffectDisplay.sprite(sprite, [priority], [anim_speed], [x_origin], [y_origin]) -> Struct  
```

Returns a EffectDisplaySprite struct.

**Parameters**  
Parameter | Type | Description
| - | - | -
`sprite` | sprite | The sprite ID.
`[priority]` | number | The [draw priority](https://github.com/ReturnsAPI/ReturnsAPI/wiki/EffectDisplay#DrawPriority) . <br>`EffectDisplay.DrawPriority.ABOVE` by default.
`[anim_speed]` | number | The animation speed. <br>`0` by default.
`[x_origin]` | number | The x coordinate of the origin (offset). <br>`0` by default.
`[y_origin]` | number | The y coordinate of the origin (offset). <br>`0` by default.

<br><br>
