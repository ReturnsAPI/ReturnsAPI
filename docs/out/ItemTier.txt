## Static Methods  

<a name="new"></a>
```lua
ItemTier.new(identifier) -> ItemTier  
```

Creates a new item tier with the given identifier if it does not already exist,
or returns the existing one if it does.

**Parameters**  
Parameter | Type | Description
| - | - | -
`identifier` | string | The identifier for the item tier.

<br><br>

<a name="find"></a>
```lua
ItemTier.find(identifier, [namespace]) -> ItemTier or nil  
```

Searches for the specified item tier and returns it.
If no namespace is provided, searches in your mod's namespace first, and vanilla tiers second.

**Parameters**  
Parameter | Type | Description
| - | - | -
`identifier` | string | The identifier to search for.
`[namespace]` | string | The namespace to search in.

<br><br>

<a name="wrap"></a>
```lua
ItemTier.wrap(tier) -> ItemTier  
```

Returns an ItemTier wrapper containing the provided item tier.

**Parameters**  
Parameter | Type | Description
| - | - | -
`tier` | number | The item tier to wrap.
