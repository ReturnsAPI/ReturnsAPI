## Static Methods

<a name="new"></a>
```lua
Packet.new() -> Packet
```

Creates a new Packet and returns it.  

**Parameters**  
None

<br><br>

<a name="wrap"></a>
```lua
Packet.wrap(packet_id) -> Packet
```

Returns a Packet wrapper containing the provided packet ID.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`packet_id` | number | The packet ID to wrap.

<br><br>

## Instance Methods

<a name="set_serializers"></a>
```lua
packet:set_serializers(serializer, deserializer) -> nil
```

Set the serialization and deserialization functions for the packet.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`serializer` | function | The serialization function.
`deserializer` | function | The deserialization function.

<br><br>

<a name="send_to_all"></a>
```lua
packet:send_to_all([...]) -> nil
```

Sends a packet message to all clients.  
Must be called as host.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`[...]` |  | *Optional.* A variable number of arguments to pass to the serialization function.

<br><br>

<a name="send_direct"></a>
```lua
packet:send_direct(target, [...]) -> nil
```

Sends a packet message to a specific client.  
Must be called as host.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`target` |  | The target player to send to.
`[...]` |  | *Optional.* A variable number of arguments to pass to the serialization function.

<br><br>

<a name="send_exclude"></a>
```lua
packet:send_exclude(target, [...]) -> nil
```

Sends a packet message all clients *except* a specific one.  
Must be called as host.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`target` |  | The target player to exclude.
`[...]` |  | *Optional.* A variable number of arguments to pass to the serialization function.

<br><br>

<a name="send_to_host"></a>
```lua
packet:send_to_host([...]) -> nil
```

Sends a packet message to the host.  
Must be called as client.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`[...]` |  | *Optional.* A variable number of arguments to pass to the serialization function.

<br><br>
