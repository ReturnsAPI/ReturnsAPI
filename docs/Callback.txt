<a name="add"></a>
```lua
Callback.add(callback, fn, [priority]) -> number
```

Registers a function under a callback type.  
Returns the unique ID of the registered callback.  
  
**Convention**  
To allow for a decent amount of space between priorities,  
use the enum values in [`Callback.Priority`](https://github.com/ReturnsAPI/ReturnsAPI/wiki/Callback#Priority).  
If you need to be more specific than that, try to keep a distance of at least `100`.  
  
**Parameters**  
Parameter | Type | Description
| - | - | -
`callback` | number | The [callback type](https://github.com/ReturnsAPI/ReturnsAPI/wiki/Callback#constants) to register under.
`fn` | function | The function to register. <br>The parameters for it depend on the callback type (see below).
`[priority]` | number | *Optional.* The priority of the function. <br>Higher values run before lower ones; can be negative. <br>`0` by default.
  
**Callbacks**  
Parameters are listed in order for each callback.  
Callback                            | Parameters  
| --------------------------------- | ----------  
`ON_LOAD`                           | None  
`POST_LOAD`                         | None  
`ON_STEP`                           | None  
`PRE_STEP`                          | None

`ON_SECOND`                         | `minute` (number) - Current minute on the timer <br>`second` (number) - Current second on the timer

<a name="remove"></a>
```lua
Callback.remove(id) -> nil
```

Removes a registered callback function.  
The ID is the one from [`Callback.add`](https://github.com/ReturnsAPI/ReturnsAPI/wiki/Callback#add).  

**Parameters**  
Parameter | Type | Description
| - | - | -
`id` | number | The unique ID of the registered callback to remove.

<br><br>

<a name="remove_all"></a>
```lua
Callback.remove_all() -> nil
```
