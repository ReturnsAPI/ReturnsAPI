## Static Methods

<a name="get_type_name"></a>
```lua
Callback.get_type_name(num_id) -> string
```

Returns the string name of the callback type with the given ID.  

**Parameters**  
Parameter | Type | Description
| - | - | -
`num_id` | number | The numerical ID of the callback type.

<br><br>

<a name="add"></a>
```lua
Callback.add(callback, fn, [priority]) -> number
```

Registers a function under a callback type.  
Returns the unique ID of the registered callback.  
  
**Convention**  
To allow for a decent amount of space between priorities,  
use the enum values in [`Callback.Priority`](https://github.com/ReturnsAPI/ReturnsAPI/wiki/Callback#Priority).  
If you need to be more specific than that, try to keep a distance of at least `100`.  
  
**Parameters**  
Parameter | Type | Description
| - | - | -
`callback` | number | The callback type to register under.
`fn` | function | The function to register. <br>The parameters for it depend on the callback type (see below).
`[priority]` | number | *Optional.* The priority of the function. <br>Higher values run before lower ones; can be negative. <br>`0` by default.
